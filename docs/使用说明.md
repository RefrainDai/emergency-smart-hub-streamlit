# 使用说明

本文面向第一次使用本平台的评审和演示人员，按 Windows 电脑说明操作；macOS / Linux 可参考相同命令。

## 1. 解压安装包

1. 将作品安装包解压到一个不含特殊权限限制的目录，例如桌面或下载目录。
2. 进入解压后的 `CaseCache-UAV` 文件夹。
3. 确认目录中包含 `app.py`、`requirements.txt`、`demo_data`、`assets`、`scripts` 等内容。

## 2. 安装 Python

1. 推荐安装 Python 3.10 或 3.11。
2. 可以从 Python 官网下载安装包。
3. 安装时勾选 “Add Python to PATH”。
4. 安装后打开终端，输入：

```bash
python --version
```

能看到版本号即表示安装成功。

## 3. 打开终端

Windows 推荐使用 PowerShell：

1. 在 `CaseCache-UAV` 文件夹空白处按住 Shift 并点击鼠标右键。
2. 选择“在终端中打开”或“在此处打开 PowerShell”。
3. 确认终端当前路径位于 `CaseCache-UAV`。

## 4. 安装依赖

在终端中运行：

```bash
pip install -r requirements.txt
```

如果下载较慢，可以稍后重试，或切换到稳定网络环境。

## 5. 启动平台

在终端中运行：

```bash
streamlit run app.py
```

浏览器通常会自动打开页面。若没有自动打开，请手动访问：

```text
http://localhost:8501
```

Windows 也可以双击或运行：

```bat
scripts\run_windows.bat
```

macOS / Linux 可以运行：

```bash
bash scripts/run_unix.sh
```

## 6. 加载示例数据

1. 打开“多模态任务接入”页面，可先查看智能体工具调用轨迹如何转化为任务负载。
2. 如需使用内置任务负载，打开“灾害任务接入”页面。
3. 选择“使用内置演示任务负载”。
4. 页面会自动展示任务类型数量、请求序列数量、请求到达量曲线等信息。

## 7. 运行一次完整演示

推荐顺序如下：

1. **系统总览**：查看项目定位、核心流程和指标卡片。
2. **多模态任务接入**：生成演示智能体工具调用轨迹，并可将其转化为任务负载。
3. **灾害任务接入**：确认示例任务负载已加载。
4. **历史案例库**：选择内置历史决策轨迹，点击“构建/刷新历史案例库”。
5. **相似案例检索**：选择任务序列和当前时隙，查看 Top-K 相似历史案例。
6. **ICL缓存决策**：选择“ICL少样本缓存决策（主方法）”，点击“运行预测”。
7. **调度效果评估**：点击“生成策略对比”，查看不同方法的当前演示结果。
8. **单步可解释分析**：选择一个时隙，查看状态、相似案例和反馈评价。
9. **结果导出**：导出运行结果、策略对比结果、单步决策解释和演示报告摘要。

## 8. 导出结果

进入“结果导出”页面后，可以下载：

- `演示_运行结果.csv`
- `演示_策略对比结果.csv`
- `演示_单步决策解释.csv`
- `演示_历史案例表.csv`
- `演示_报告摘要.md`

导出的 CSV 默认使用中文列名，便于直接放入演示材料或答辩附件。

## 9. 上传文件格式与参数说明

如果需要上传自己的任务负载文件或历史决策轨迹，请先阅读 `docs/数据格式与参数说明.md`。该文档说明了任务负载文件支持的 `CSV`、`JSON`、`JSONL` 格式，历史决策轨迹的推荐 `JSONL` 结构，以及每个页面可调参数的含义。

## 10. 演示视频解说稿

如需录制网页演示视频，可直接参考 `docs/网页演示完整解说稿.md`。该脚本按照“系统总览 → 灾害任务接入 → 历史案例库 → 相似案例检索 → ICL缓存决策 → 调度效果评估 → 单步可解释分析 → 结果导出”的顺序编写。

## 11. 常见错误与解决方案

**提示 `streamlit` 不是内部或外部命令**  
先运行 `pip install -r requirements.txt`。如果仍失败，可以尝试：

```bash
python -m streamlit run app.py
```

**浏览器打不开页面**  
确认终端中 Streamlit 正在运行，然后访问 `http://localhost:8501`。

**案例库页面没有案例**  
确认选择了内置历史决策轨迹，并点击“构建/刷新历史案例库”。如果上传自定义文件，需要保证轨迹长度足够。

**ICL缓存决策页面提示需要案例库**  
先去“历史案例库”页面构建案例库，再回到该页面运行。

**外部 ICL 推理模型接口未配置**  
这是正常情况。平台会使用本地相似案例缓存预测完成演示，不影响基本流程。
