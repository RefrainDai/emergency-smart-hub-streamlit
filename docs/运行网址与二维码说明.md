# 运行网址与二维码说明

## 1. 本地运行地址

平台本地启动后，浏览器通常访问：

```text
http://localhost:8501
```

该地址只适合当前电脑访问。

## 2. 局域网演示地址

Streamlit 启动后，终端中通常会显示 Local URL 和 Network URL。若同一局域网内的其他设备需要访问，应使用 Network URL。

常见形式类似：

```text
http://192.168.x.x:8501
```

使用前请确认：

- 电脑和手机或展示设备处于同一局域网；
- 防火墙允许访问该端口；
- 运行平台的电脑没有休眠或断网。

## 3. 二维码使用建议

二维码只适合指向局域网或公网可访问地址。生成二维码前，应先在目标设备上手动打开该地址确认可访问。

不要把以下地址作为公网二维码提交：

```text
http://localhost:8501
```

原因是 `localhost` 只代表当前设备本机，其他设备扫描后无法访问你的电脑。

## 4. 公网链接建议

如果正式提交需要公网访问链接，建议将平台部署到：

- 云服务器；
- 校内可访问服务器；
- Streamlit Community Cloud；
- 其他支持 Python 与 Streamlit 的托管平台。

部署前请确认演示数据、图片资源和依赖文件完整，不要上传 `.env`、API Key 或个人隐私文件。

## 5. 简单二维码生成方式

1. 启动平台。
2. 获取可访问的网址。
3. 在手机或另一台电脑上验证该网址能打开。
4. 使用常见二维码生成工具生成二维码。
5. 在提交材料中标注该二维码适用范围，例如“仅限同一局域网访问”。
