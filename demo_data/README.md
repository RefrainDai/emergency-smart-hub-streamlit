# 演示数据说明

本目录提供平台内置演示数据，用于展示灾害响应场景下多模态任务服务的资源缓存预测流程。

## 文件内容

- `sample_requests.csv`：灾害任务请求负载样例，共 1200 条记录，覆盖 5 类任务类型与 5 条请求序列。
- `sample_metatrain_history.jsonl`：历史决策轨迹样例，共 2500 条记录，用于构建历史案例库。

## 灾害任务类型

样例数据中的任务类型已替换为灾害响应业务语境：

- 快速问答服务
- 受灾目标识别
- 结构化报告生成
- 灾害图像深度分析
- 危险源态势研判

样例请求序列已替换为灾害事件序列，例如地震救援、洪涝巡检、山火监测、台风灾情和危化事故等场景。

## 字段说明

平台内部仍使用兼容字段名读取数据：

- `log_time`：时间步
- `num`：请求到达量
- `access_key_id`：请求序列
- `game_name`：任务类型

用户界面和导出结果会将这些字段显示为中文正式术语。保留内部字段名是为了确保现有数据读取、分组、案例检索和评估流程稳定运行。

## 使用建议

默认演示可直接使用本目录数据。若上传自定义任务负载或历史决策轨迹，请参考 `docs/数据格式与参数说明.md`。
